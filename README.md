Handmade Product Price and Demand Prediction

📌 Project Overview

This project uses Machine Learning to predict the optimal price and demand of handmade products based on various features like material cost, labor hours, product type, customer type, and season.

🚀 Features

* Price Prediction using Random Forest Regressor
* Demand Prediction using Random Forest Classifier
* Data preprocessing using Pandas
* Visualization using Matplotlib

🛠️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* Google Colab

📊 Dataset

The dataset is manually created and contains features such as:

* Material Cost
* Labor Hours
* Product Type
* Customer Type
* Season

⚙️ How It Works

1. Data preprocessing and encoding
2. Model training using Random Forest
3. Prediction of price and demand
4. Visualization of results

▶️ How to Run

1. Open the project in Google Colab
2. Upload the dataset (data.csv)
3. Run all cells
4. View predictions and graphs

📈 Output

* Predicted Price
* Demand (High / Low / Medium)
